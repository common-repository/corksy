== Description ==
Corksy is an ecommerce solution for the winery industry. 

Whether you are launching a new winery or taking an existing winery online, corksy provides powerful features to do e-commerce right. Any themes suited for your business can be used with this plugin to create an efficient store front giving customers great user experience.  

= All Features =
* Store front 
* Viewing products by collection
* Signing up or checkout as guest 
* Signup for subscription 
* Book wine tasting event
* order management 
* Profile management 
* Reward points program integration 
* Quick signup for payment processing 
* Verifying shipment compliance to all states of USA
